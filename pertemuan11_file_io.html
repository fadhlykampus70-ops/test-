Apa itu File I/O?

Selama ini program kita hanya menyimpan data di dalam memori RAM — begitu program ditutup, semua data hilang. Dengan File I/O (Input/Output), kita bisa menyimpan data ke dalam file di harddisk supaya bisa dibaca kembali kapan saja.

💡 Analogi Sederhana

Bayangkan file seperti buku catatan. Kamu bisa membukanya, menulis di dalamnya, membaca isinya, lalu menutupnya kembali. Python punya fungsi bawaan untuk melakukan semua itu.

Fungsi utama yang akan kita pakai adalah open() bawaan Python:

  Sintaks dasar open()
PYTHON 
# Sintaks:
file = open("nama_file.txt", "mode")

# Contoh membuka file untuk ditulis:
f = open("catatan.txt", "w")
f.write("Halo, ini file pertamaku!")
f.close()  # Jangan lupa tutup!
⚠️ Perhatian Penting

Setiap kali membuka file dengan open(), kamu wajib menutupnya dengan .close(). Jika tidak, file bisa rusak atau data tidak tersimpan sempurna. Kita akan belajar cara yang lebih aman menggunakan with nanti.

2

Mode Membuka File

Parameter kedua di open() adalah mode — menentukan apa yang mau kamu lakukan dengan file itu.

"r"
Read (Baca)
Hanya membaca isi file. File harus sudah ada, kalau tidak error.
"w"
Write (Tulis)
Menulis ke file. Jika file sudah ada, isinya dihapus dulu!
"a"
Append (Tambah)
Menambah tulisan di akhir file tanpa menghapus isi sebelumnya.
"x"
Create (Buat Baru)
Membuat file baru. Error jika file sudah ada.
"r+"
Read + Write
Bisa baca dan tulis sekaligus. File harus sudah ada.

Menulis ke File

Gunakan method .write() untuk menulis teks ke file. Jika ingin pindah baris, tambahkan karakter \n.

🧪 Coba di Thonny!
  contoh_tulis.py
THONNY 
# ===== MENULIS FILE =====
# File akan dibuat di folder yang sama dengan script ini

f = open("biodata.txt", "w")

f.write("Nama : Budi Santoso\n")
f.write("Usia : 20 tahun\n")
f.write("Kota : Pekanbaru\n")

f.close()

print("✅ File berhasil dibuat!")
✅ Hasil yang Diharapkan

Setelah program dijalankan, file biodata.txt akan muncul di folder yang sama dengan file Python-mu. Buka file tersebut, isinya akan berisi tiga baris biodata.

Bisa juga menulis banyak baris sekaligus menggunakan .writelines() dengan list:

🧪 Coba di Thonny!
  writelines_contoh.py
THONNY 
# Tulis banyak baris sekaligus dengan writelines()

daftar_buah = [
    "Mangga\n",
    "Jeruk\n",
    "Durian\n",
    "Rambutan\n"
]

f = open("buah.txt", "w")
f.writelines(daftar_buah)
f.close()

print("✅ Daftar buah tersimpan!")

Membaca Isi File

Ada 3 cara membaca file di Python, masing-masing cocok untuk situasi berbeda:

MethodMembacaHasilKapan Dipakai

🧪 Coba di Thonny!
  baca_read.py — cara 1: .read()
THONNY 
# Pastikan biodata.txt sudah dibuat dari contoh sebelumnya!
# .read() → baca seluruh isi file sekaligus

f = open("biodata.txt", "r")
isi = f.read()
f.close()

print("Isi file:")
print(isi)
🧪 Coba di Thonny!
  baca_readlines.py — cara 3: .readlines()
THONNY 
# .readlines() → hasilnya LIST, tiap baris jadi satu elemen

f = open("buah.txt", "r")
baris = f.readlines()
f.close()

print("Daftar buah:")
for i, buah in enumerate(baris, 1):
    # .strip() untuk hapus \n di akhir baris
    print(f"  {i}. {buah.strip()}")
💡 Tips: .strip()

Saat membaca file, setiap baris biasanya diakhiri dengan karakter \n (newline). Gunakan .strip() untuk menghapusnya supaya tampilan lebih bersih.

5

Menambah Data ke File (Append)

Mode "a" menambahkan tulisan di akhir file tanpa menghapus isi yang sudah ada. Sangat berguna untuk membuat log atau catatan harian.

🧪 Coba di Thonny!
  append_contoh.py
THONNY 
# Jalankan beberapa kali — perhatikan data terus BERTAMBAH!

import datetime

waktu_sekarang = datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")

f = open("log_akses.txt", "a")
f.write(f"Diakses pada: {waktu_sekarang}\n")
f.close()

print(f"✅ Log ditambahkan! Waktu: {waktu_sekarang}")

# Baca kembali untuk membuktikan
f = open("log_akses.txt", "r")
print("\n📋 Isi log sekarang:")
print(f.read())
f.close()
🔍 Yang Harus Kamu Amati

Jalankan kode di atas 3–4 kali. Setiap kali dijalankan, isi log_akses.txt akan bertambah satu baris baru dengan timestamp berbeda. Inilah perbedaan mode "a" vs "w"!

6

Cara Aman: Pakai with

Cara yang paling direkomendasikan saat bekerja dengan file adalah menggunakan keyword with. Keunggulannya: file otomatis ditutup meskipun terjadi error di tengah program.

✅ Kenapa pakai with?

Dengan with open(...) as f:, Python otomatis memanggil f.close() ketika blok kode selesai dieksekusi — bahkan jika ada error. Tidak perlu ingat tutup file manual!

🧪 Coba di Thonny!
  with_menulis.py
THONNY 
# ===== MENULIS dengan WITH =====
with open("nilai.txt", "w") as f:
    f.write("Nama   : Siti\n")
    f.write("Nilai  : 92\n")
    f.write("Grade  : A\n")
# ↑ file sudah otomatis tertutup di sini

print("✅ Data tersimpan!")

# ===== MEMBACA dengan WITH =====
with open("nilai.txt", "r") as f:
    isi = f.read()

print("📄 Isi file nilai.txt:")
print(isi)

Perbandingan cara lama vs cara baru:

  ❌ Cara Lama
f = open("file.txt", "r")
isi = f.read()
# kalau error di sini,
# f.close() tidak dipanggil!
f.close()
  ✅ Cara Baru (with)
with open("file.txt", "r") as f:
    isi = f.read()
# otomatis tertutup,
# bahkan jika ada error!

Integrasi dengan UI Tkinter

Di pertemuan sebelumnya kamu sudah belajar membuat GUI dengan Tkinter. Sekarang kita gabungkan! Contoh berikut membuat aplikasi Buku Catatan Sederhana — pengguna bisa menulis di text area, lalu simpan/baca dari file.

🖥️ Fitur Aplikasi yang Akan Dibuat

✅ Text area untuk menulis catatan  |  ✅ Tombol Simpan ke file  |  ✅ Tombol Buka dari file  |  ✅ Label status informasi

🧪 Coba di Thonny!
  buku_catatan_ui.py — Aplikasi Buku Catatan dengan File I/O
THONNY + TKINTER 
import tkinter as tk
from tkinter import messagebox, filedialog

# ===== FUNGSI SIMPAN =====
def simpan_file():
    # Ambil isi text area
    isi = text_area.get("1.0", tk.END)
    
    if isi.strip() == "":
        messagebox.showwarning("Peringatan", "Catatan masih kosong!")
        return
    
    # Dialog pilih lokasi simpan
    path = filedialog.asksaveasfilename(
        defaultextension=".txt",
        filetypes=[("Text Files", "*.txt"), ("All Files", "*.*")]
    )
    
    if path:  # Jika user tidak cancel
        with open(path, "w", encoding="utf-8") as f:
            f.write(isi)
        label_status.config(text=f"✅ Tersimpan: {path}", fg="#28c840")

# ===== FUNGSI BUKA =====
def buka_file():
    path = filedialog.askopenfilename(
        filetypes=[("Text Files", "*.txt"), ("All Files", "*.*")]
    )
    
    if path:
        with open(path, "r", encoding="utf-8") as f:
            isi = f.read()
        
        # Hapus isi lama, masukkan isi baru
        text_area.delete("1.0", tk.END)
        text_area.insert(tk.END, isi)
        label_status.config(text=f"📂 Dibuka: {path}", fg="#5b7fff")

# ===== FUNGSI HAPUS =====
def hapus_area():
    text_area.delete("1.0", tk.END)
    label_status.config(text="🗑️ Area catatan dikosongkan.", fg="gray")

# ===== BUAT JENDELA UTAMA =====
root = tk.Tk()
root.title("📝 Buku Catatan Sederhana")
root.geometry("600x450")
root.configure(bg="#f0f4f8")

# Header
header = tk.Label(root, text="📝 Buku Catatan",
    font=("Arial", 14, "bold"), bg="#5b7fff", fg="white",
    pady=10)
header.pack(fill="x")

# Frame tombol
frame_btn = tk.Frame(root, bg="#e8ecf0", pady=8)
frame_btn.pack(fill="x", padx=10)

tk.Button(frame_btn, text="📂 Buka File", command=buka_file,
    bg="#5b7fff", fg="white", padx=12, pady=5,
    relief="flat", cursor="hand2").pack(side="left", padx=5)

tk.Button(frame_btn, text="💾 Simpan File", command=simpan_file,
    bg="#28c840", fg="white", padx=12, pady=5,
    relief="flat", cursor="hand2").pack(side="left", padx=5)

tk.Button(frame_btn, text="🗑️ Hapus", command=hapus_area,
    bg="#ff5f5f", fg="white", padx=12, pady=5,
    relief="flat", cursor="hand2").pack(side="left", padx=5)

# Text area
frame_text = tk.Frame(root, bg="#f0f4f8")
frame_text.pack(fill="both", expand=True, padx=10, pady=10)

scrollbar = tk.Scrollbar(frame_text)
scrollbar.pack(side="right", fill="y")

text_area = tk.Text(frame_text,
    font=("Consolas", 12),
    bg="white", fg="#1e2333",
    wrap=tk.WORD,
    yscrollcommand=scrollbar.set,
    relief="flat", bd=5)
text_area.pack(fill="both", expand=True)
scrollbar.config(command=text_area.yview)

# Label status
label_status = tk.Label(root,
    text="💡 Mulai ketik catatan atau buka file yang sudah ada.",
    font=("Arial", 9), bg="#e8ecf0", fg="gray",
    anchor="w", padx=10, pady=5)
label_status.pack(fill="x")

root.mainloop()
🎯 Penjelasan Kode UI di Atas

• filedialog.asksaveasfilename() → membuka dialog "Save As" seperti di aplikasi nyata
• filedialog.askopenfilename() → membuka dialog "Open File"
• text_area.get("1.0", tk.END) → ambil semua teks dari baris 1 karakter 0 sampai akhir
• encoding="utf-8" → agar teks Indonesia (huruf beraksara) tersimpan dengan benar

Latihan Mandiri

Kerjakan latihan berikut di Thonny untuk memperkuat pemahaman:

🧩 Latihan 1 — Buku Nilai Sederhana

Buat program yang meminta input nama siswa dan nilai (loop sampai user ketik "selesai"), lalu simpan seluruh data ke file nilai_siswa.txt. Setelah disimpan, baca kembali dan tampilkan isinya.

🧪 Coba di Thonny!
  latihan1_nilai.py
LATIHAN 
# ===== LATIHAN 1: Buku Nilai =====

data_siswa = []

print("=== Input Nilai Siswa ===")
print("Ketik 'selesai' untuk berhenti.\n")

while True:
    nama = input("Nama siswa : ")
    if nama.lower() == "selesai":
        break
    nilai = input("Nilai      : ")
    data_siswa.append(f"{nama} - {nilai}\n")

# Simpan ke file
with open("nilai_siswa.txt", "w", encoding="utf-8") as f:
    f.writelines(data_siswa)

print("\n✅ Data tersimpan!")

# Baca dan tampilkan kembali
print("\n📋 Rekapitulasi Nilai:")
print("-" * 30)
with open("nilai_siswa.txt", "r", encoding="utf-8") as f:
    for baris in f:
        print(" ", baris.strip())
