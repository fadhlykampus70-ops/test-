Alignment Gambar dengan Float
float-left

Gambar dengan float-left akan membuat teks mengalir di sebelah kanan gambar. Ini berguna untuk layout artikel atau berita yang memerlukan gambar di sisi teks. Gunakan clearfix pada parent agar layout tidak rusak setelah gambar float.

float-right

Gambar dengan float-right akan membuat teks mengalir di sebelah kiri gambar. Kombinasi float dan clearfix adalah teknik layout klasik yang tetap relevan digunakan bersama Bootstrap.

10E — Screen Reader class="sr-only"

Class sr-only menyembunyikan elemen secara visual tetapi masih dapat dibaca oleh screen reader (teknologi bantu untuk penyandang disabilitas visual). Ini penting untuk aksesibilitas web.

Teks ini terlihat secara normal oleh semua pengguna.

Teks ini HANYA terbaca oleh screen reader, tidak terlihat di layar.

Kalimat di atas sudah ada tapi tidak terlihat (gunakan DevTools untuk membuktikannya — ada elemen dengan class sr-only).

sr-only menggunakan teknik CSS: position: absolute; width: 1px; height: 1px; overflow: hidden; sehingga elemen ada di DOM tapi tidak terlihat secara visual.
Lewati ke konten utama

sr-only-focusable: tombol "skip to content" yang muncul hanya saat di-tab (coba tekan Tab di halaman ini).

10F — Responsive Utility: Menampilkan & Menyembunyikan Konten d-none / d-block / d-sm-none / d-md-block ...

Bootstrap 4 menggunakan utility class d-* untuk mengatur visibilitas elemen di berbagai ukuran layar. Format: d-[breakpoint]-[nilai]. Breakpoint: sm, md, lg, xl. Nilai: none, block, inline, inline-block, flex, dll.

Class Efek
d-none Sembunyikan di semua ukuran layar
d-block Tampilkan (display: block) di semua ukuran layar
d-none d-md-block Sembunyi di layar kecil, tampil ab layar md ke atas
d-block d-md-none Tampil di layar kecil, sembunyi ab layar md ke atas
d-none d-lg-block Sembunyi di bawah lg, tampil di lg ke atas
Demo Live — Ubah ukuran browser untuk melihat efeknya:
d-block d-md-none — Elemen ini HANYA terlihat di layar kecil (HP/tablet kecil, lebar < 768px)
d-none d-md-block — Elemen ini HANYA terlihat di layar sedang ke atas (tablet besar / desktop, lebar ≥ 768px)
d-none d-lg-block — Elemen ini HANYA terlihat di layar besar ke atas (desktop, lebar ≥ 992px)
d-block — Elemen ini selalu terlihat di semua ukuran layar
10G — Float Utility & Image Replacement float-left / float-right / text-hide
Float Utility

Class float-left, float-right, dan float-none untuk mengatur posisi elemen. Tersedia juga versi responsif seperti float-md-right.

float-left float-right
float-none (normal flow)
Image Replacement (text-hide)

Class text-hide digunakan untuk menggantikan teks dengan gambar latar (background-image) tanpa mengorbankan aksesibilitas — teks tetap ada di HTML untuk screen reader, namun tidak terlihat secara visual.

Teknik ini umum digunakan pada logo atau elemen branding yang ditampilkan sebagai gambar tetapi perlu teks untuk SEO dan aksesibilitas.
